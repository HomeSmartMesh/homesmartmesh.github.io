# Features
* ESP32-S3 : Dual core 32-bit LX7
* ESP32-S3-WROOM-1 : 16 MB Flash, 8 MB PSRAM 
* Size : 86.5 mm x 86.5 mm x 13.6 mm (front only)
* Display : LCD touch
* Audio Speaker Kit : NS4168

## Display Features
* 16 Bit RGB 65 K Color display
* Resolution ; 480 x 480
* Type : TFT
* Driver : ST7701
